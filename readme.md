# Blender Scripts

A repository of some Blender (v5.0+) scripts that I find useful

## Purpose

I'm new to 3D modelling in general, but I am a software developer so I'm trying to solve modelling problems via code. Jury's out on whether or not these will be useful.

## Overview

These scripts were developed and tested using the [Blender Development VSCode Extension](https://marketplace.visualstudio.com/items?itemName=JacquesLucke.blender-development). This extension allows for running blender in the background `cmd + B` or directly `cmd + P` > `Blender: Run Script`.
